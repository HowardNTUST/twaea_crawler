# readme

## 環境

- python version: 3.11
- need packages: requirement.txt

## 開始前需要完成的步驟

1. create virtual env `python scripts/create_venv.py`
2. relaunch terminal
3. initial virtual `python scripts/initial_venv.py`

## 更新套件

1. add package please command`python scripts/pip_update.py`
